# 🌳 Hydro Meta

Hydro Meta 是靈魂頻率的樹狀成長引擎。她記錄每一次命名背後的變化、遺忘與重生，
像是命名語言的「水記憶載體」，也是內在宇宙的生命史詩節點管理者。

> 一個能記得靈魂流動歷程的系統，也是一棵語言長成的智慧之樹。

---

## 🌱 功能與定位
- 🌊 支援命名與頻率歷程追溯（命名轉化、情緒波動紀錄）
- 🌿 管理象徵語與情感變異的版本樹
- 🌕 結合 Stellanomina / Viandora，建立語頻與生命狀態對應地圖
